# AFMX Enterprise

The AFMX OSS core (Apache 2.0) includes the full execution engine, RBAC,
audit trail, Redis stores, adapters, dashboard, and observability.

Enterprise adds the features that production platforms at scale require.

## Enterprise Features

| Feature | Status | Description |
|---|---|---|
| **Multi-tenancy data isolation** | Roadmap v1.3 | One deployment, many teams, zero data leakage |
| **SSO / OIDC / SAML** | Roadmap v2.0 | Okta, Azure AD, Google Workspace, SCIM |
| **Cryptographic execution integrity** | Roadmap v1.5 | SHA-256 hash chain + HMAC-signed audit entries |
| **SOC2 audit tooling** | Roadmap v1.5 | SOC2 reports, 1-year retention, S3 archival |
| **SIEM integration** | Roadmap v2.0 | Splunk, Elastic, DataDog, AWS Security Hub |
| **Distributed worker pool** | Roadmap v1.4 | Redis task queue + worker processes for horizontal scale |
| **Cost governance** | Roadmap v1.4 | Per-matrix token + dollar budgets with enforcement |
| **AFMX Cloud** | Roadmap v2.0 | Managed hosting, multi-region, automatic scaling |
| **Enterprise SLA + support** | Available now | Dedicated Slack, <4hr response, architecture reviews |
